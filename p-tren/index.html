Apa itu P-Tren?

Sebuah Program Repeat Order, dimana mitra dapat melakukan pembelanjaan sejumlah paket produk dan mendapatkan pembagian keuntungan dari omset yang diperoleh dari Program Repeat Oder tersebut. Program ini berlaku mulai Agustus 2018 sampai dengan Desember 2018.

Program Repeat Order ini dijual Rp. 1.250.000,- (Include PPN 10%) , dengan jenis paket produk pilihan sebagai berikut :

PAKET 1
Lihat Produk
PAKET 2
Lihat Produk
PAKET 3
Lihat Produk
